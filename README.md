> Flask application scaffolding tool

## Installation

First, install [Yeoman](http://yeoman.io) and generator-new-flask-app using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-new-flask-app
```

Then generate your new project:

```bash
yo new-flask-app
```
